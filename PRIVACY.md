# Privacy Policy for ImgtoPrompt

Last updated: April 2026

## 1. Overview

ImgtoPrompt is a browser extension that helps users analyze images (such as Pinterest images) and generate descriptive prompts using AI.

We are committed to protecting your privacy. This extension does **not collect, store, or sell any personal data**.

---

## 2. Data We Access

ImgtoPrompt only processes the following data:

- **Website Content (Images):**
  The extension reads images from the current webpage (e.g., Pinterest) only when the user actively triggers the analysis.

- **User Input:**
  Any input provided by the user (e.g., clicking “analyze” or “enhance”) is processed temporarily.

---

## 3. How Data Is Used

The accessed data is used solely to:

- Analyze the selected image
- Generate AI-based prompts
- Improve prompt quality (e.g., enhancement feature)

All processing is done **only to provide the core functionality of the extension**.

---

## 4. Data Transmission

To generate results, image data may be sent to a secure backend service and AI APIs (such as Google Gemini API).

- Data is transmitted securely (HTTPS)
- No personal identity is attached
- Data is **not stored permanently**

---

## 5. Data Storage

ImgtoPrompt does **not store**:

- Personal information
- Browsing history
- Images or prompts on any server

Temporary data may be processed in memory and discarded immediately after use.

---

## 6. Data Sharing

We do **not**:

- Sell user data
- Share user data with third parties for advertising
- Use data for tracking or profiling

Data is only used to fulfill the requested feature.

---

## 7. Permissions Usage

ImgtoPrompt may request the following permissions:

- **Active Tab / Host Permissions**
  Used to access images on the current webpage for analysis

- **Context Menus (if enabled)**
  Allows users to trigger analysis via right-click

- **Storage**
  Used only to store local preferences (e.g., UI settings)

These permissions are strictly limited to core functionality.

---

## 8. User Control

Users have full control:

- The extension only runs when triggered
- No background tracking
- No automatic data collection

---

## 9. Security

We take reasonable measures to protect data:

- Secure HTTPS communication
- No persistent storage of user data
- Minimal data handling principle

---

## 10. Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be reflected with a new "Last updated" date.
